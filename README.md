# This is an app that manages sportives, trainers and teams.
- CRUD opeartions
- Filters
- Sorts
- GUI

Technologies that were used:
- Back-end:
  - Java Spring
  - PostgreSQL
- Front-end:
  - Angular

- For the web server I used Apache Tomcat.

Note: This is a project made for a laboratory from the faculty.
